Sistema de Gestión Integral para Óptica

Una solución completa (web y móvil) para la administración y el análisis de datos.

El Reto

Una empresa de óptica necesitaba modernizar y unificar sus operaciones. Sus procesos de gestión de inventario, pedidos, clientes y empleados estaban descentralizados y eran ineficientes. Además, carecían de herramientas para analizar sus datos de ventas y tomar decisiones estratégicas.

Mi Solución

Desarrollé una plataforma integral que consiste en una aplicación web y una móvil. El sistema centraliza la gestión de inventarios, pedidos a proveedores, base de datos de clientes y registros de empleados. La funcionalidad estrella es una página de analítica que procesa los datos de ventas y genera visualizaciones claras sobre el rendimiento de productos y las relaciones con proveedores y clientes.

Tecnologías Utilizadas

  • Desarrollo Web (HTML, CSS, JavaScript)
  • Desarrollo Móvil (Android/Kotlin)
  • Backend y Base de Datos (SQL)
  • Visualización de Datos y Analítica
