App de Visualización de Inventario

Una aplicación nativa para Android conectada a la nube para gestionar clientes y visualizar datos.

El Reto

Para una empresa ficticia, el objetivo era crear una solución móvil que permitiera a sus empleados acceder y gestionar datos de clientes e inventario desde cualquier lugar, asegurando que la información estuviera siempre sincronizada y presentada de forma clara.

Mi Solución

Desarrollé una aplicación nativa para Android con Kotlin. La app cuenta con un sistema de registro y gestión de clientes y usuarios. Su funcionalidad principal es un dashboard que se conecta a una base de datos en la Nube para obtener datos del inventario y presentarlos a través de gráficos y listas intuitivas, permitiendo una toma de decisiones rápida y basada en datos actualizados.

Tecnologías Utilizadas

  • Kotlin
  • Android Studio & Android SDK
  • XML para Layouts
  • Servicios en la Nube (Cloud Database, como Firebase)
