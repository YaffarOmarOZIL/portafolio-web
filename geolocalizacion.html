Sistema de Geolocalización IoT con ESP32

Un proyecto integral que combina hardware y software para rastreo en tiempo real.

El Reto

Crear un dispositivo de rastreo de bajo costo y alta eficiencia. El principal desafío técnico consistía en integrar la programación de hardware embebido con el desarrollo web para lograr una visualización de datos fluida y en tiempo real.

Mi Solución

Construí un dispositivo físico utilizando un microcontrolador ESP32 y un módulo GPS. Programé el firmware en C++ para capturar coordenadas de geolocalización y transmitirlas vía Wi-Fi a un servidor. En el lado del software, desarrollé una página web con JavaScript que recibe estos datos y los muestra al instante en un mapa interactivo, permitiendo un seguimiento en vivo del dispositivo.

Tecnologías Utilizadas

  • ESP32 Microcontroller
  • Arduino (C++)
  • JavaScript y WebSockets
  • HTML5 y CSS3
  • Internet de las Cosas (IoT)
